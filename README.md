## Project Description
The Learn Track Student and Course Management System is a Java-based application designed to manage students, courses, and enrollments. It provides functionality to add, update, and deactivate students and courses, as well as manage enrollments for students in courses. The system is built with clean code principles and follows an interface-driven design for better scalability and maintainability.
